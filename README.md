## Installation

```
poetry install
```

```
cp .env.example .env
```

Заполнить переменные в .env
```
SERVER_GS4U_URL=https://www.gs4u.net/en/s/358042.html
TELEGRAM_BOT_TOKEN=<YOUR_TOKEN>
CHAT_ID=<YOUR_CHAT_ID>
SERVER_NAME=
```


## Docker


Необходимо создать .env и запустить: 
```
docker-compose up -d
```
